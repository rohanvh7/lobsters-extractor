## Lobsters Feed extractor with feedparser and rich

# Usage
usage: uv run main.py [-h] [-t Tag-Name] [-l]

This program is written to extract lobste.rs articles for a perticular tag

options:
  -h, --help        show this help message and exit
  -t T, -tagname T
  -l, -list
